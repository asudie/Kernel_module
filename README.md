# Kernel_module

### 1. To compile do
> make

### 2. To load do
> sudo insmod my_module.ko

### 3. To unload do
> sudo rmmod my_module.ko

### 4. To see my_module among modules
> sudo lsmod

### 5. To see output in logs
> sudo dmesg | tail
